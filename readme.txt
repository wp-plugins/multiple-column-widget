You will be able to put a list of posts in a single row

== Description ==

This widget helps you to put posts in a row. The width for each post will be divided evenly with the total number of post you have selected during the creation of widget.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download and extract the plugin
2. Upload folder `multiple_column_widget/` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create widget specific posts on 'Multiple Column Widget Post'
5. Assign the post to the widget on 'Multiple Column Widget Post/Create Widget'
6. Activate the widget on 'Appearance/Widgets' menu
